# Schwarzinger.dev - Persönliche Website

Diese statische Website stellt mich vor, meinen Beruf und mein Hobby. Das Design ist modern im Dark Mode mit sanften Animationen für eine ansprechende Benutzererfahrung.

## Anpassung

1. Öffnen Sie `index.html` in einem Browser, um die Seite zu sehen.
2. Bearbeiten Sie die Platzhalter in `index.html`:
   - [Ihr Vorname Nachname]
   - [Ihr Beruf]
   - [Ihr Ort]
   - [Beschreibung Ihrer Erfahrung]
   - [Ihre Fähigkeiten auflisten]
   - [Einige Projekte erwähnen]
   - [Ihr Hobby]
3. Ersetzen Sie die Platzhalter-Bilder mit Ihren eigenen Bildern (z.B. in einem `assets/` Ordner).
4. Passen Sie `styles.css` an, um das Design zu ändern.

## Hosting

Da es statisch ist, können Sie es auf GitHub Pages, Netlify oder einem anderen statischen Host hosten.

## Technologien

- HTML5
- Tailwind CSS
- Custom CSS für Animationen
